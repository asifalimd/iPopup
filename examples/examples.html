Integration Steps

1. Copy the directory "popup" to your project 2. Include the CSS file after all CSS files. <link rel="stylesheet" href="popup/css/style.css"> 3. Include the JS file after all JS file includes, make sure its added after jQuery <script type="text/javascript" src="popup/js/popup.js"></script> 4. To open a popup when clicking on any link/button just add css class "open-popup" and add popup div ID class="open-popup" data-popup-id="<Your popup ID>" 5. You need not to hide the popup explicitly using display:none property. The plugin hides all popups, given in "data-popup-id", on pageload. 6. To close a popup, add css class "close-popup" to close buttons/links. It automatically closes the active popup. 7. To enable auto close icon on top right of popup just add the below custom attribute to your popup link data-has-close-icon="true"

Popup Examples

Big Popup with close button inside |   View Code |   Close Button Code
<a class="open-popup" data-popup-id="popup01" >Big Popup with close button inside</a>
<input type="button" class="close-popup" value="Close Me">


Small Popup with close icon |   View Code
<a class="open-popup" data-popup-id="popup02" data-has-close-icon="true" >Small Popup with close icon</a>


Image Popup with close icon |   View Code
<a class="open-popup" data-popup-id="popup03" data-has-close-icon="true" >Image Popup with close icon</a>
